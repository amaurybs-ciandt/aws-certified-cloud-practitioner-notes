## aws organizations
aws organization sé um serviço de gerenciamento de contas que permite organizar e consolidar as contas da aws de uma org, você pode gerenciar a parte de faturamento, segurança e compatibilidade.

- gerenciamento centralizado de todas as contas
- faturamento consolidado para todas as contas membros.
- agrupamento hierarquico das contas
- politicas para centralizar o controle dos serviços com SCP (serviço de politicas para org), tags e backups.
- integração com o serviço de IAM
- integração com outros serviços da aws
